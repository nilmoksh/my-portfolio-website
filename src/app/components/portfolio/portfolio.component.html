REST API Development

We developer api using restful webservices .we cater to business where their front end application needs to create data driven applications

Restful webservice are created using sprintboot framework . we use spring core libraries ,spring security and spring transaction management.We use AWS cloud for our deployment.

JSON Web Token Security

JSON Web Tokens, commonly known as JWTs, are tokens that are used to authenticate users on applications. our client applications that use JWTS no longer have to hold cookies or other session data about their users. This characteristic facilitates scalability while keeping applications secure.

For Authentication and authorization in our application we use Spring Security and Redis Storage.

Data Persistence

Java Persistence API is our choice for data persistance. We can also work with different ORM solutions in case there is a requirement.

We have lot of experience in fine tuning the application using JPA . Using Spring Boot we configure annotation based java persistence. We also work with second level cache like EhCache in case of multipage application
